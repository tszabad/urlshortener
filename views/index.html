Tom's URL Shortener Microservice

Usage

  1. You can enter a URL to [project_url]/api/shorturl/new and you will receive a shortened URL in the JSON response.
    Example : {"original_url":"www.google.com","short_url":VzwMqoJP1}
  2. If you pass an invalid URL that doesn't follow the usual Url format, the response will give you an error.
  3. When you visit the shortened URL, it will redirect you to your original link.
